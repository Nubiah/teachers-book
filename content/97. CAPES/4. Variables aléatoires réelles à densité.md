# Introduction
De tout temps, les humains ont utilisé les probabilités, ne serait-ce que pour décider d'aller ou non chasser dans tel ou tel endroit. il y avait déjà une évaluation des risques. Les mathématiques permettent d'étudier les risques précisément et de prendre des décisions éclairées.
Le manuel *The Doctrine of Chances* publiés en 1718 par Abraham de Moivre est le premier ouvrage sur la théorie des probabilités. Il est le traité de référence jusqu'en 1912 quand *Théorie analytique des probabilités* est publié par Pierre-Simon Laplace. Cet ouvrage est précurseur par l'utilisation des fonctions dans le cadre des probabilités. A la même époque, Carl Friedrich Gauss travaille sur la théorie des erreurs dans les mesures physiques. Il introduit une loi de probabilité continue, la loi normale.
Dans cette leçon, nous donnerons quelques exemples de variables aléatoires à densité.
## Pré-requis
- Variables aléatoires, espérance, variance, écart type, loi binomiale

# I. Introduction
**Remarque** : Si X est une variable aléatoire suivant un loi à densité, alors X peut prendre une infinité non dénombrable de valeurs et $\forall x \in \mathbb{R}, P(X=x)=0$
On va donc s'intéresser à des probabilités telles que :
- $P(X\leq x)$
- $P(x_{1}\leq X\leq x_{2})$ où $(x_{1},x_{2})\in \mathbb{R}^2$ et $x_{1}<x_{2}$
**Remarque 2** : $P(x_{1}\leq X\leq x_{2})=P(X\leq x_{2})-P(X\leq x_{1})$ et $P(X<x_{1})=P(X\leq x_{1})$
> [!warning] **Définition**
> On dit qu'une fonction $f$, définie sur $\mathbb{R}$ est une *fonction de densité* si $f$ est continue sauf en un nombre dénombrable de points
> - $\forall x\in \mathbb{R}, f(x)\geq 0$
> - $\int^{+\infty}_{-\infty}f(x)dx=1$

>[!warning] **Définition 2**
>Soit $f$ une fonction de densité. On appelle *fonction de répartion* la fonction $F$ définie par $F(x)=\int^{x}_{-\infty}f(t)dt$
>**Remarque** : si $f$ est la fonction de densité d'une certaine variable aléatoire X alors $F(x)=P(X\leq x)$

> [!warning] **Définition 3**
> Soit X une variable aléatoire continue de fonction de densité $f$.
> - On appelle *espérance* de X le réel noté E(X), défini par $E(X)=\int^{+\infty}_{-\infty}tf(t)dt$ s'il existe
> - On appelle *variance* de X le réel noté V(X), défini par $V(X)=\int^{+\infty}_{-\infty}(t-E(X))^2 f(t)dt$ s'il existe
> - On appelle *écart type* de X le réel noté $\sigma(X)$, défini par $\sigma(X)=\sqrt{ V(X) }$ s'il existe

>[!note] **Propriétés**
>- *Linéarité de l'espérance*
>  Soient X et Y deux variables aléatoires et $a$ un réel.
>  Alors, sous réserve d'existence : $E(aX+Y)=aE(X)+E(Y)$
>- On a aussi $V(aX)=a^2V(X)$ et, si X et Y sont indépendantes, $V(X+Y)=V(X)+V(Y)$
>- *Formule de König-Huygens*
>  Soit X une variable aléatoire continue de fonction de densité $f$. Sous réserve d'existence, on a $V(X)=E(X^2)-E(X)^2$

**Remarque** : l'espérance d'une variable aléatoire correspond à la valeur de la moyenne pondérée des valeurs qu'elle peut prendre.
Plus la variance est élevée, plus la variable aléatoire peut prendre des valeurs loin de son espérance. Une variable aléatoire avec une variance de 0 par exemple prendrait toujours la même valeur. 
# II. Majoration de l'écart à la moyenne
>[!note] **Propriété : Inégalité de Bienaymé-Tchebychev**
>Pour toute variable aléatoire X et pour tout $a>0$, on a : $P(|X-E(X)|\geq a)\leq \frac{V(X)}{a^2}$

>[!warning] **Définition**
>Soit $n\in \mathbb{N}^*$ et $(X_{i})_{i\in[1,n]}$,$n$ variables aléatoires indépendantes et de même loi qu'une variable aléatoire X. On appelle la *variable aléatoire moyenne* la variable aléatoire $M_n$ définie par $M_{n}=\frac{1}{n}\sum^n_{i=1}X_{i}$

> [!note] **Propriétés**:
> - *Inégalité de concentration*
>   Soit $M_n$ la variable aléatoire définie ci-dessus, alors, pour tout $a>0$ : $P(|M_{n}-E(X)|\geq a)\leq \frac{V(X)}{n^2}$
> - *Loi des grands nombres*
>   Pour tout $a>0$, $\lim_{ n \to +\infty }P(|M_{n}-E(X)|\geq a =0)$

**Remarque** : Ces propriétés majorent l'écart d'une variable aléatoire ou d'une moyenne à la moyenne. Elles ont l'avantage d'être vraies pour n'importe quelle variable aléatoire X. Cependant, la majoration est souvent grossière.
La loi des grands nombres nous garantit que, en simulant un nombre assez important de fois une même expérience aléatoire, la moyenne obtenue peut être aussi proche que l'on veut de l'espérance théorique, ce qui est pratique lorsque le phénomène est trop compliqué pour être étudié théoriquement. On le simule des milliers de fois par informatique et on a une bonne idée de l'espérance théorique.

## Remarque générale sur les lois à densité
Avec les variables aléatoires à densité, calculer une probabilité revient à calculer une intégrale. il est possible de trouver, grâce à un algorithme, une valeur approché de probabilité cherchée.
*Exemple* :
```python
from math import exp
from math import sqrt
from math import pi
from random import random

#Approche la proba que a<X<b ou X suit une loi N(m,s) par la méthode de monté carlo

s=1
m=0

def f(x):
	return exp(-.5*(((x-m)/s)**2))/(s*sqrt(2*pi))

def monte(a,b):
	if a > b:
		a.b=b.a
	cpt = 0
	for _ in range(1000000):
		x = random()*(b-a)+a
		y = random()
		if y < f(x):
			cpt += 1
	return cpt/1000000*(b-a)

a = -1000
b = 1
print(monte(a,b))
```
